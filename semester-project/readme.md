# Oivaly - Modern E-Commerce Storefront

**Oivaly** is a modern, responsive e-commerce storefront built as a semester project for the Browser Programming course at Savonia University of Applied Sciences. It features a clean, minimal design with interactive shopping capabilities, dynamic product filtering, and custom SVG-generated product art.

## ✨ Features

* **Dynamic Product Art:** Product images are generated dynamically using inline SVGs, completely removing the need for external image hosting for placeholders.
* **Category Filtering & Search:** Users can instantly filter products by categories (Men, Women, Accessories) or use the search bar to find specific items.
* **Interactive Shopping Cart:** Real-time cart counter with toast notifications when items are added.
* **Quick View Modal:** Click on any product to open a detailed modal view without leaving the main page.
* **Responsive Design:** A custom, inline-styled responsive layout that works beautifully on mobile, tablet, and desktop screens.
* **Wishlist functionality:** Users can "like" products to save them to a localized wishlist.

## 🛠️ Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (React)
* **Language:** TypeScript
* **Styling:** Custom Inline CSS & Flexbox/CSS Grid
* **Backend Integration:** Configured to support [Medusa](https://medusajs.com/) for scalable backend commerce operations.
* **Deployment:** Vercel
