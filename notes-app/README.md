## Project Overview
This project is a very simple full-stack Notes App. The purpose of this assignment is to practice the complete workflow of building and deploying a web application with a frontend, backend, database, and public online hosting.

The app allows the user to:
- add a note
- view all notes
- delete a note

The application is kept simple. No login system or advanced styling is included.

---

## Goal of the Assignment
The goal of this assignment is to learn how a full-stack application works from end to end:

**Frontend → Backend API → Database → Deployment**

At the end of the assignment, the app must be available online through a public URL.

---

## Technologies Used
This project follows the recommended technology stack:

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js + Express
- **Database:** Supabase (PostgreSQL)
- **Frontend Hosting:** Vercel
- **Backend Hosting:** Render
- **Code Repository:** GitHub
