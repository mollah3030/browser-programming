# Lecture 03 – CSS Basics

## What I implemented in this lecture
- I used an external CSS file to style my portfolio page while keeping the main HTML structure unchanged.
- I added spacing with margin and padding to make the layout cleaner and easier to read, and I used `max-width` to control the content width.
- I also designed the navigation as a horizontal menu by removing the default list bullets and link underlines.

## CSS selectors I used (at least 5)
- Element selector (`body`, `h1`, `img`)
- Group selector (`header, footer`)
- Descendant selector (`nav ul`, `nav li`)
- ID selector (`#main`)
- Pseudo-class (`a:hover`)

## One thing I struggled with
- One thing I found difficult was deciding when to use margin and when to use padding for spacing between elements.
